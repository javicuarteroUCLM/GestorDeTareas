# Introducción
Repositorio para el laboratorio de GSI donde realizaremos un sistema de gestión de tareas eficiente

## ¿En qué consiste?
Se trata de una aplicación de escritorio desarrollada 100% con Python, tanto como para el backend como para la interfaz gráfica (se ha usado la librería Tkinter), donde cada usuario puede gestionar sus tareas y clasificarlas según el tipo de tarea (Ocio, Trabajo, Cotidiana), también cuenta con un calendario integrado para poder fijar una fecha de inicio o fin a esa tarea.

La aplicación también cuenta con dos tipos de rol: Jefe, Empleado.
  -Un empleado solo puede gestionar (añadir, editar, eliminar) sus tareas.
  -Un jefe puede gestionar sus tareas, y además la de los empleados, es decir, puede añadir, editar o eliminar las tareas de otros usuarios.
